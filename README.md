# 🧠 Intelligent Contract (GenLayer-style)

## Overview
This project demonstrates a simple Intelligent Contract that integrates AI decision-making with verifiable execution.

## Features
- AI-based decision engine
- Deterministic verification layer
- API interface (FastAPI)
- Transparent logging system

## How it works
1. User submits request
2. AI evaluates input
3. Contract stores result
4. Verifier re-checks consistency
